# Advent of Code 2025

Solutions for Advent of Code 2025 in Python.

## Structure

- `src/aoc_2025/day_XX/`: Solutions for each day
  - `main.py`: The solution script
  - `input.txt`: Puzzle input
  - `example.txt`: Example input
- `src/aoc_2025/utils/`: Utility functions

## Running

To run a day's solution:

```bash
cd src/aoc_2025/day_XX
python main.py
```

## Requirements

- Python 3.8+
- Dependencies listed in `pyproject.toml`
